This is my version of simpbms for bmw csc modules earlier SP26 ones 
designed to run on the audino due and supporting 2 can buses so a manximim of 12csc slaves,
renumbering of modules per bus is retained so any combination of modules can be used,
global numbering system employed based on bus and module id so there are no id chashesacross buses

support currently direct HV and outlander charger
